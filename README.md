# Course Advisory & Recommendation System

## 📌 Overview
This system helps students select appropriate courses based on academic records, prerequisites, and program requirements.

## 🎯 Features
- Student registration & login
- Course management
- Academic record tracking
- Course recommendation engine

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## 📅 Development Plan
10-week structured development using Git workflow.
